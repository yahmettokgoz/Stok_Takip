# Ekran Görüntüleri

Bu klasöre projenin ekran görüntülerini ekleyin.

## Gerekli Ekran Görüntüleri:

1. **login.png** - Login/Giriş ekranı
2. **dashboard.png** - Ana sayfa dashboard (5 kart gösterimi)
3. **ai-chatbot.png** - AI Asistan sohbet ekranı
4. **urunler.png** - Ürünler listesi
5. **satis.png** - Satış formu
6. **kritik-stok.png** - Kritik stok uyarıları
7. **raporlar.png** - Raporlar sayfası (opsiyonel)
8. **ayarlar.png** - Ayarlar sayfası (opsiyonel)

### Ekran Görüntüsü Alma:
1. Uygulamayı çalıştırın
2. İlgili sayfayı açın
3. `Win + Shift + S` ile ekran görüntüsü alın
4. Bu klasöre kaydedin
5. Dosya isimlerini yukarıdaki gibi adlandırın

### Boyut Önerileri:
- Format: PNG
- Max genişlik: 1920px
- Optimize edilmiş boyut (< 500KB)
